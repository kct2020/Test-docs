## Markdown

[Markdown It](https://github.com/markdown-it/markdown-it#syntax-extensions) - the vscode markdown renderer. supports tables and ~~strike-through~~  
[Markdown All in One](https://github.com/yzhang-gh/vscode-markdown#features) - vscode plugin that extends the editor and preview more closely to GFM  
[GitHub Flavored Markdown Spec](https://github.github.com/gfm/) - Subset of CommonMark Markdown specification  
[CommonMark Spec](https://spec.commonmark.org/current/) - a base specification for markdown
